# Velotur Yug

Стартовый отдельный проект сайта для `veloturyug.ru`.

## Что внутри

- `Next.js` c `App Router`
- первая версия главной страницы
- структура, которую можно развивать в туры, FAQ, блог и бронирование

## Запуск

```bash
npm install
npm run dev
```

После этого сайт будет доступен на `http://localhost:3000`.
